# Git操作



| 命令                                  | 操作             | 说明                 |
| ------------------------------------- | ---------------- | -------------------- |
| git remote -v                         | 查看当前远端地址 |                      |
| git remote set-url origin {remoteUrl} | 修改当前远端地址 | 可以为SSH或者HTTP(S) |
|                                       |                  |                      |
